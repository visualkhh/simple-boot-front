simple-boot-cli
===
# sbf
```shell
npm install simple-boot-front-cli
```

## serve
* http server and proxy
  * --path        dist path
  * --port        server port
  * --proxy       proxy url
  * --watch       fileChange browser refresh

## create
* create simple-boot-front template project

## example
```shell
sbf serve --path ../dist --watch
```
